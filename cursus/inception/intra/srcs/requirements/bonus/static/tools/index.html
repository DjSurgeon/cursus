Docker Cheatsheet

Containers Lifecycle

  • docker ps -a List all containers (running & stopped)
  • docker exec -it <name> sh Access a running container
  • docker logs -f <name> Follow container logs in real-time

Compose Commands

  • docker compose up -d --build Build and run in detached mode
  • docker compose down -v Stop and remove containers + volumes
  • docker compose --profile bonus up Run specific profiled services

Cleanup & Maintenance

  • docker system prune -a --volumes Remove all unused containers, networks, images, and volumes (Use with caution)
  • docker volume ls List all local volumes
