# 42 Cursus Projects

This repository contains my personal implementations of the projects in the 42 School common core curriculum. Each project is designed to master specific concepts in C, C++, and system administration.

## 🚀 Projects Overview

The projects are organized within the `cursus/` directory:

### Foundation (C)
- **[libft](./cursus/libft/):** My first custom C library, re-implementing standard libc functions and more.
- **[get_next_line](./cursus/get_next_line/):** A function that reads and returns a line from a file descriptor.
- **[ft_printf](./cursus/ft_printf/):** A custom implementation of the standard `printf` function.

### Algorithms & Logic
- **[push_swap](./cursus/push_swap/):** Sorting data on a stack with a limited set of instructions and the lowest possible number of moves.
- **[philo](./cursus/philo/):** Solving the Dining Philosophers problem to learn about threads, mutexes, and shared memory.

### Graphics & Visualization
- **[fdf](./cursus/fdf/):** A 3D wireframe representation of a landscape using the MinilibX library.

### System & Infrastructure
- **[born2beroot](./cursus/born2beroot/):** System administration project focused on setting up a virtual machine with Debian, following strict security protocols.
- **[netpractice](./cursus/netpractice/):** Solving networking problems related to IP addressing, subnets, and routing.
- **[inception](./cursus/inception/):** System administration with Docker, setting up a complex infrastructure with multiple services.

### Inter-process Communication
- **[minitalk](./cursus/minitalk/):** A small client/server communication program using UNIX signals (SIGUSR1/SIGUSR2).

### C++ Modules
- **[cpp](./cursus/cpp/):** A series of modules focused on learning Object-Oriented Programming (OOP) in C++.

### Networking & Servers
- **[irc](./cursus/irc/):** Implementation of an IRC server from scratch in C++.

### Testing & Utilities
- **[test](./cursus/test/):** A collection of test suites and scripts used to verify the correctness of the projects.
